# **App Name**: SyllabiQ

## Core Features:

- Mock User Registration/Login: Interactive mock screens for signup, login, and profile management to simulate user authentication flow.
- Dashboard Simulation: Simulate displaying available grades, subjects, and topics with interactive elements.
- Interactive Content Access: Simulate access to each topic including text summaries, images, videos, key points, and important questions using placeholders. All elements will be interactive.
- Search & Filters Simulation: Interactive simulation of searching topics by name, subject, or chapter; and filters for subject or chapter.
- Bookmarks/Favorites Simulation: Ability to save topics interactively (no backend storage).
- Contact/Support Simulation: A simple interactive form or chat placeholder to test the user interaction.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to evoke calmness and focus, ideal for study environments.
- Background color: Light grey (#F5F5F5) to provide a neutral and clean backdrop, reducing eye strain during long study sessions.
- Accent color: Muted purple (#B39DDB) for interactive elements like buttons and links, adding a touch of creativity without being distracting.
- Body and headline font: 'PT Sans', a humanist sans-serif combining a modern look with approachability and warmth. Its clarity and legibility makes for long stretches of pleasurable reading, whether headlining or body text.
- Use a consistent set of simple, line-style icons for navigation and topic categories.
- Design a fully responsive layout optimized for both desktop and mobile, with a focus on comfortable and intuitive navigation on smaller screens.
- Implement smooth, subtle transitions and animations to enhance user interaction and provide feedback (e.g., when saving a favorite topic).
